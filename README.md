## Что это

**Windows Loader** — это настольный помощник для Windows в стиле персонального ассистента наподобие **JARVIS**.

Это не просто заготовка и не сырой dev-проект. Это уже собранный и подготовленный desktop loader с голосовым управлением, готовой средой запуска, удобным интерфейсом и базовой логикой для повседневного использования на ПК.

Windows Loader рассчитан на то, чтобы быть помощником прямо на рабочем столе:
- быстро запускаться;
- быть всегда под рукой;
- принимать голосовые команды;
- помогать с повседневными действиями на компьютере;
- работать как удобный desktop companion, а не как обычное окно программы.

---

## Текущее состояние проекта

На текущем этапе Windows Loader — это **JARVIS-подобный голосовой помощник для ПК**.

### В этой версии уже есть
- голосовые команды;
- управление окнами и частью функций Windows;
- быстрый доступ к системным действиям;
- запуск приложений и сайтов голосом;
- управление громкостью и медиа;
- авторизация и профиль пользователя;
- готовый установщик и подготовленная среда запуска.

### В этой версии пока нет
- полноценной встроенной нейросети;
- AI-диалогов как у современных LLM-ассистентов;
- полноценного режима “умной беседы” в стиле ChatGPT/Claude/GPT.

Иными словами, **сейчас это не нейросетевой чат-ассистент**, а именно:
- голосовой помощник;
- помощник для управления ПК;
- JARVIS-ориентированный desktop loader;
- база для дальнейшего расширения проекта.

## What It Is

**Windows Loader** is a desktop assistant for Windows, designed in the style of a personal helper similar to **JARVIS**.

This is not just a raw developer build or an unfinished prototype. It is a prepared desktop loader with voice control, a ready-to-run environment, a convenient interface, and core desktop helper logic for everyday PC usage.

It is designed to feel like a companion on your desktop:
- quick to launch;
- always nearby;
- able to accept voice commands;
- useful in everyday PC workflows;
- working like a desktop companion instead of just another window.

---

## Current Project State

At the current stage, Windows Loader is a **JARVIS-style voice assistant for PC**.

### Already included
- voice commands;
- control of windows and selected Windows actions;
- quick access to system-level operations;
- opening apps and websites by voice;
- media and volume control;
- user authorization and profile;
- ready installer and prepared runtime setup.

### Not included yet
- a full built-in neural network;
- advanced AI chat capabilities;
- full LLM-style conversation mode.

In other words, **this version is not yet a neural network assistant**.
Right now it is primarily:
- a voice-based desktop assistant;
- a PC control helper;
- a JARVIS-oriented loader;
- a strong base for future expansion.

---

## What It Can Do

Windows Loader is made for users who want:
- a personal desktop assistant on Windows;

- a prepared system without complex manual setup.

### Core features
- voice-based system control;
- app and website launching;
- window management;

- persistent desktop companion behavior.
